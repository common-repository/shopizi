Woocommerce infinity scroll your products at your awesome site!

== Description ==
This plugin lets you infinity scroll your Woocommerce products to your WordPress site.

== Installation ==
1. Download the plugin.
2. Activate the plugin in the WordPress Admin Panel.
3. Usage of the plugin:

 Shortcode: "[shopizi selector="my_products" category="laptops" method="post" action="shopizi" post_type="product" ppp="5" order="DESC"]

 or in php place do_shortcode('[shopizi selector="my_products" category="laptops" method="post" action="shopizi" post_type="product" ppp="5" order="DESC"]')".

 The "selector" is ID of the your DOM element.

 NO NEED  functions.php !
